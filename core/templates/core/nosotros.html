Sobre nosotros

Perricompras.cl se crea en 2022, todo a partir de las redes sociales, para poder ayudar a una fundacion
sin fines de lucro, siendo este nuestro objetivo principal decidimos poner mas de nosotros creando este sitio web
para que asi ustedes tambien puedan ayudar a esta fundacion!.

Mision

Agregar valor global a nuestros clientes entregando mejores valores a través de esta pagina web , con
la mejor calidad dentro del mercado de accesorios para perros.

Vision

Mantener una atención rápida, amigable y preferencial a todos nuestros clientes, en lo que refiere a las
ventas y consultas generadas por nuestra plataforma.
